## Tools 🛠️
- [Python](python.org)
- Libraries
    - smtp
    - python-decouple
    - customtkinter
    - tkinter
    - threading
    - os

## How to run?
```bash
    # In your git bash, clone the repo.
    git clone https://github.com/angelo-francisco/Email-Sender.git

    # Activate the enviroment
    # att.: can be different on another OS
    ./venv/Scripts/Activate.ps1

    # run the project
    python ./src/main.py
```
